# Iris Flower Classification

## Project Overview
This project is a basic Machine Learning classification model that predicts the species of Iris flowers using flower measurements.

The model classifies flowers into three species:
- Iris-setosa
- Iris-versicolor
- Iris-virginica

The project uses the **Scikit-learn** library to train and test the machine learning model.

## Dataset
The dataset used is the **Iris Dataset**, which contains flower measurements such as:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

Dataset Link:
https://www.kaggle.com/datasets/saurabh00007/iriscsv/data

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Google Colab / Jupyter Notebook

## Machine Learning Model
This project uses the **Random Forest Classifier** to classify Iris flower species.

## Project Steps
1. Load the Iris dataset
2. Prepare input and output data
3. Split dataset into training and testing data
4. Train the machine learning model
5. Predict flower species
6. Evaluate model performance using accuracy and classification report

## Model Performance
The model achieved approximately **100% accuracy (1.0)** on test data.

Evaluation Metrics:
- Accuracy
- Precision
- Recall
- F1-Score
