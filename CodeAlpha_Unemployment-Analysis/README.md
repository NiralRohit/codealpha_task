# Unemployment Analysis with Python

## Project Overview
This project focuses on analyzing unemployment rate data in India using Python. The aim is to understand unemployment trends, investigate the impact of Covid-19, identify seasonal patterns, and generate insights that may help in economic and social policy-making.

## Objectives
- Analyze unemployment rate data.
- Perform data cleaning and preprocessing.
- Explore unemployment trends using visualizations.
- Study the impact of Covid-19 on unemployment rates.
- Identify seasonal or monthly patterns.
- Present insights and recommendations.

## Dataset
The project uses the following datasets:

1. **Unemployment in India.csv**
2. **Unemployment_Rate_upto_11_2020.csv**

These datasets contain unemployment-related statistics across different regions and time periods in India.

## Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly

## Project Workflow
1. Import required libraries
2. Load datasets
3. Data cleaning and preprocessing
4. Exploratory Data Analysis (EDA)
5. Visualization of unemployment trends
6. Covid-19 impact analysis
7. State-wise unemployment analysis
8. Seasonal trend analysis
9. Conclusion and insights

## Key Features
- Data cleaning and preprocessing
- Trend visualization
- Covid-19 unemployment analysis
- Region-wise comparison
- Monthly trend analysis
- Correlation analysis using heatmaps

## Results
- Unemployment rates varied significantly across regions.
- Covid-19 caused a sharp increase in unemployment during 2020.
- Seasonal trends were identified through monthly analysis.
- Certain states experienced consistently higher unemployment rates.
