# CodeAlpha Machine Learning Projects

This repository contains four Machine Learning and Data Analysis projects completed as part of the CodeAlpha Internship Program using Python and Google Colab.

## Projects Included

### 1. Iris Flower Classification

A machine learning classification project used to predict the species of Iris flowers based on flower measurements.

#### Objectives

* Use Iris flower measurements as input features
* Train a classification model to predict flower species
* Evaluate model accuracy and performance

#### Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn

#### Dataset Features

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

#### Target Classes

* Setosa
* Versicolor
* Virginica

---

### 2. Car Price Prediction with Machine Learning

A regression-based machine learning project used to predict car prices using multiple car-related features.

#### Objectives

* Analyze car-related features
* Train regression models for price prediction
* Perform preprocessing and model evaluation

#### Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn

#### Evaluation Metrics

* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* Root Mean Squared Error (RMSE)
* R² Score

---

### 3. Unemployment Analysis with Python

#### Objectives

#### Technologies Used

* Python

## Installation

Clone the repository:

```bash
git clone <your-repository-link>
```

Move into the project directory:

```bash
cd CodeAlpha-Projects
```

Install required libraries:

```bash
pip install -r requirements.txt
```
